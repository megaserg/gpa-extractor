GPA Extractor
=============
Compute your GPA from SPbSU diploma supplement PDF!

Made in 2013.

How to run
----------
  ```
  $ ./pdf2gpa.sh supplement.pdf 
  File supplement.txt: total subjs: 61, total marks: 37, GPA: 4.891892
  ```
